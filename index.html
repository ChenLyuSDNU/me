Research

  • Trustworthy & Efficient Code Generation. High-performance, low-memory code generation; safety-aware decoding; white-box auditable pipelines (constrained decoding, rollback, candidate ranking, efficiency/security evaluation, evidence chains).
  • Repo-level Code Security. Vulnerability detection at repository scale; LLM‚ÜíCodeQL rule generation, repair, and evaluation; joint coverage √ó quality metrics and benchmarks.
